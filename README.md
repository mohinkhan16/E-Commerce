# 🛒 ShopCart - E-Commerce Website

A modern and responsive **E-Commerce Website** built using **HTML, CSS, Bootstrap, and JavaScript**. This project demonstrates core JavaScript concepts like DOM Manipulation, Local Storage, CRUD Operations, Shopping Cart Management, and Bootstrap Modals.

---

## 🚀 Features

- 🛍️ Display Products
- ➕ Add New Product
- ✏️ Update Existing Product
- 🗑️ Delete Product
- 🛒 Add Products to Cart
- ➕ Increase Quantity
- ➖ Decrease Quantity
- ❌ Remove Product from Cart
- 💰 Grand Total Calculation
- ✅ Checkout Functionality
- 💾 Local Storage Support
- 📱 Responsive Design
- 🌙 Modern Dark Glassmorphism UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Local Storage API

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/shopcart.git
```

### 2. Open Folder

```bash
cd shopcart
```

### 3. Run Project

Simply open

```
index.html
```

in your browser.

No installation required.
